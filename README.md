# smart-home
collection of smart home hacks as a means to give back to the awesome optolink-splitter community

Key contributions:
## Viessmann Optolink Connection to OpenHAB
These files contain my existing findings related to the undocumented Vitotronic 200 (KW2) addresses needed to read from my oil heating system.
Down the road, I plan to contribute also to making the optolink-splitter more accessible to openHAB users.

## Hermes Agent (successor to OpenClaw) integration with OpenHAB
This is a skill for Hermes Agent (see: https://hermes-agent.nousresearch.com/) which allows to use Hermes Agent to control your smart home rather than Google or Alexa. Once LLMs move to become fully local (we should see this in a few weeks or months) this will be a fabulous privacy enhancement to manage both your openHAB server and your smart home
